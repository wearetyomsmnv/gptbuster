# gptbuster
Webdir fuzzer based on ChatGPT

# usage

'git clone https://github.com/wearetyomsmnv/gptbuster'
'cd gptbuster'
'pip3 install -r requirements'

'python3 main.py <website> <api-key>'

ex: python3 main.py https://google.com/ sk***********
